About This Map

This web map displays crime data for the City of Albuquerque. Click (or tap) on the map to see crimes within half a mile of your click location. Data is for the last 180 days (roughly 6 months).


Quick Tutorial

The map starts with a view of Albuquerque. Clicking or tapping the map inside the city limits will zoom to and display crimes within half a mile of that location. Click on an individual crime marker to list the type of crime and when it occurred. I usually want to focus on three crimes - auto burglary, auto theft, burglary/breaking and entering. They are displayed with orange markers. All other crimes are displayed in blue. Click the checkbox in the bottom left if you only want to view auto and burglary crimes. You can change the basemap via the icon in the top right. Press it to select from two different street basemaps or an aerials basemap.


Geek Stuff

I did this web map for fun, as a proof of concept based on Paul Crickard's excellent blog post Query Albuquerque Open Data with Turf.js. Check it out if you are interested in what is going on behind the scenes. You can also visit the Leaflet and Turf web sites for more info about these two great open source GIS resources.

More could be done to expand and improve this web map. I may or may not add to it in the future. Please remember it was done for fun rather than to build a fully featured app. Feel free to grab the code and build on it yourself!
